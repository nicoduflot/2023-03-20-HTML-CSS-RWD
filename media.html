Les media en HTML

Les vidéos hébergées sur le site

On utilise la balise <video>

La source de la vidéo est directement indiquée dans l'attribut src, attention, si le navigateur n'est pas capable de lire la vidéo, elle ne s'affiche pas.

Si on dispose de plusieurs format de la même vidéo (dimension, extension), on peut indiquer plusieurs sources, le navigateur choisira celle qui est la plus adaptée.

Les vidéos hébergées sur un site tier

Des sites tels que Youtube et dailymotion donnent la possibilité d'incruster leurs vidéos sur nos sites, en utilisant la balise iframe.
L'iframe est une balise qui permet d'appeler une page d'un autre site (ou du site actuel) dans un cadre délimité.
Il faut toutefois que le site ciblé autorise les iframes
